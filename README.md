# 温度系列刮刮乐 H5 原型

手机端互动广告原型。双击 `index.html` 即可直接体验，无需安装依赖。

## 已实现流程

1. 营销首屏点击“开始刮草”
2. 六关全屏动态刮刮乐：动态草、糖葫芦、书本、花朵、森林、星星
3. 刮开一定区域后发现角色卡片，点击进入角色详情
4. 每个角色有不同动作和独立提示音，整体带舒缓背景音乐
5. 关闭角色卡片后可左滑切关，也可点击底部按钮继续
6. 六关完成后进入盲盒购买引导页

## 文件说明

- `index.html`：页面结构
- `styles.css`：手机端视觉、动效和安全区适配
- `app.js`：刮草画布、六关流程、角色动作和 Web Audio 音效
- `assets/`：用户提供的手绘图片素材

## 联调说明

购买按钮当前显示原型提示。接入真实商城时，将 `app.js` 中 `#buyButton` 的点击事件替换为商城链接跳转即可。
